Detailed Description

template<typename TFixedPointSet, typename TMovingImage>
class itk::MeanReciprocalSquareDifferencePointSetToImageMetric< TFixedPointSet, TMovingImage >

Computes similarity between pixel values of a point set and intensity values in an image.

This metric computes the average squared difference between pixels in the point set and transformed point set pixels in the moving image after passing the difference through a function of type \( \frac{1}{1+ \frac{ difference^2 }{ \lambda^2 } }\). \(\lambda\) controls the capture radius of the metric. The term capture radius used here is in terms of intensity domain and not in the spatial domain.

Spatial correspondence between both images is established through a Transform.
