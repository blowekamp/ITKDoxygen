Detailed Description

template<typename TImage>
class itk::FrequencyShiftedFFTLayoutImageRegionIteratorWithIndex< TImage >

Iterator providing method GetFrequency() to retrieve the frequency associated to an index. This value is related to the specific layout of frequencies from an image in the dual (frequency) space. In this case, the layout corresponds to the output of a FastFourierTransform from FFTW library. This class is a non-const version of FrequencyShiftedFFTLayoutImageRegionConstIteratorWithIndex.
