Detailed Description

template<unsigned int VDimension = 3, typename TInput = Point<double, VDimension>>
class itk::FrustumSpatialFunction< VDimension, TInput >

Spatial function implementation of a truncated pyramid.

Implements a function that returns 0 for points inside or on the surface of a truncated pyramid, 1 for points outside the truncated pyramid.
